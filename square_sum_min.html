그래프 풀이 — f = (y − (2x + 3))2 + (y + 2)2 − 6

간격 d일 때 최솟값 = d22 − 6
f = (y − (2x + 3))2 + (y + 2)2 − 6
두 축 y = 2x + 3 과 y = −2 의 간격 d일 때, y는 중점에서 최소
최솟값 = d22 − 6
(1) x, y 실수

두 축 일치: 2x + 3 = −2 → x = −52, y = −2

간격 d = 0 → 거리 합 = 0

f(−52, −2) = 0 + 0 − 6 = −6
(2) x ≥ 0, y ≥ 0

x ≥ 0 이면 축₁ = 2x + 3 ≥ 3, 축₂ = −2 → d ≥ 5

x = 0 일 때 d = 5 최소, 중점 y = 12

f(0, 12) = 252 − 6 = 132
(3) x, y 정수

축₁ = 2x + 3 (홀수) ≠ −2 → 일치 불가

d 최소 = 1 : x = −2 (축₁ = −1) 또는 x = −3 (축₁ = −3)

중점 y = −32 (비정수) → 위아래 정수에서 동일

f(−2, −1) = f(−2, −2) = f(−3, −2) = f(−3, −3) = −5
최솟값 정리
(1) −6   (2) 132   (3) −5
